# 170616-Ethen-CreamAle

**設備**

Pico 

這批實驗

* 自動化冷卻機制
* 使用熱板冷卻法
* 無人值守運作

**麥**

* 紐西蘭pale ale 4kg

**酒花**

* 苦花：Hersbrucker 4% 27g 60min
* 香花：Cascade 7% 7g 5min

**酵母**

* S-04 約8g

常溫發酵

**流程**

16.6kg -> 12.2L左右

OG 1.06 FG 1.014 ABV 5.95 IBU 24.52 SRM 5.27

糖化效率約56%

這次測試Pico自動冷卻連動，糖化到三十分的時候產生大量泡沫導致液體無法進入麥槽，初步推估是流量不足導致的泡沫，緊急切換成桶式冷卻。過程畫面嚴重閃爍。
